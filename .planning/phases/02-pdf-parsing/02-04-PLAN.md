```typescript
import { getDocument, GlobalWorkerOptions, PasswordException, InvalidPDFException }

// Disable worker (Node.js — no worker thread needed):
GlobalWorkerOptions.workerSrc = '';

const loadingTask = getDocument({ data: new Uint8Array(buffer) });
const pdfDoc = await loadingTask.promise;
// throws PasswordException for encrypted PDFs
// throws InvalidPDFException for corrupt PDFs
```

<!-- TextItem structure (research-verified): -->
```typescript
interface TextItem {
  str: string;           // the text
  transform: number[];   // [a, b, c, d, x, y] — transform[4]=x, transform[5]=y (bottom-left origin)
  height: number;        // font size in points
  fontName: string;      // pdfjs-internal key e.g. "g_d0_f1"
  width: number;
}
// To get font family: textContent.styles[item.fontName].fontFamily
// Font name normalization:
//   1. Strip ABCDEF+ prefix: name.replace(/^[A-Z]{6}\+/, '')
//   2. Detect bold: /bold|heavy|black/i.test(strippedName)
//   3. Detect italic: /italic|oblique/i.test(strippedName)
// Fallback when fontName absent: { fontName: 'Calibri', fontSize: 22, bold: false, italic: false, color: '#000000' }
```

<!-- Page dimensions: -->
```typescript
const viewport = page.view; // [x0, y0, x1, y1]
const pageWidth = viewport[2] - viewport[0];
const pageHeight = viewport[3] - viewport[1];
```

<!-- Y-proximity clustering algorithm (5 stages from research): -->
// LINE_Y_TOLERANCE = 2.0 (points) — items within this delta share a LogicalLine
// Stage 1: Collect all TextItems for a page
// Stage 2: Sort by descending Y (transform[5] DESC), then ascending X (transform[4] ASC)
// Stage 3: Group into LogicalLine[] — new group when |currentY - item.y| > LINE_Y_TOLERANCE
// Stage 4: Classify lines:
//   - Compute medianBodyHeight across all lines on the page
//   - heading: line.maxHeight >= 1.2 * medianBodyHeight
//   - header: lines before the first heading (captured as ResumeStructure.header)
//   - body: everything else
// Stage 5: Within body lines after a heading, assign indent levels by X coordinate:
//   - indent 0 (flush left, within ~10pt of marginLeft): SectionItem title/subtitle candidate
//   - indent 1+: bullet candidate

<!-- ResumeStructureSchema from Plan 02-01 (updated): -->
```typescript
// ResumeStructure.header = HeaderLine[] — lines before first section heading
// ResumeStructure.sections = Section[] — each with heading, headingStyle, items[]
// Section.items = SectionItem[] — title, titleStyle, subtitle?, bullets[]
// Bullet.id = deterministic: "experience-0-item-0-bullet-2" (section-index-item-index-bullet-index)
```

    Implementation (GREEN):
    parsePdf algorithm:
    1. `new Uint8Array(buffer)` → pass to `getDocument({ data })`. Catch PasswordException → throw PdfEncryptedError. Catch InvalidPDFException → throw PdfCorruptError.
    2. Iterate pages (1..pdfDoc.numPages). For each page, call `page.getTextContent()`. Collect all TextItems (items with `str` property, excluding TextMarkedContent).
    3. After all pages: if totalTextItems === 0 → throw PdfScannedError.
    4. Per page: run Y-proximity clustering (LINE_Y_TOLERANCE = 2.0pt). Sort items by Y DESC, X ASC. Group into LogicalLines where |prevY - itemY| <= 2.0.
    5. Compute medianBodyHeight: take all line heights, sort, pick median. A line is a heading if maxHeight >= 1.2 * median.
    6. Lines before the first heading → ResumeStructure.header (each as HeaderLine with text + TextStyle).
    7. After first heading: group into sections. Each heading starts a new section. Body lines within a section are classified by X indent into items/bullets.
    8. Font extraction: from `textContent.styles[item.fontName]` get fontFamily. Strip `^[A-Z]{6}\+` prefix from fontName. Detect bold/italic via regex. Missing fontName → use fallback values.
    9. Page dimensions from `page.view[2] - page.view[0]` and `page.view[3] - page.view[1]`. marginLeft = min X of first body text item.
    10. Minimum viable check: if sections.length === 0 OR every section has 0 bullets → throw PdfCorruptError.
    11. Validate final object with `ResumeStructureSchema.parse(result)` — if Zod throws → throw PdfCorruptError.
    12. Return validated ResumeStructure.

    ID generation for bullets: `${sectionSlug}-${sectionIdx}-item-${itemIdx}-bullet-${bulletIdx}` where sectionSlug is the heading text lowercased with spaces replaced by dashes.

    Set `GlobalWorkerOptions.workerSrc = ''` at module top level (before any getDocument calls).

    Log font fallbacks at warn level: `console.warn('[pdf.service] font fallback applied for item:', item.str)`.
