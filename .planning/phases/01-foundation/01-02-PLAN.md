ResumeStructure field requirements (from ARCHITECTURE.md + research):
- meta: { pageWidth, pageHeight, marginTop, marginBottom, marginLeft, marginRight } — all numbers (PDF points)
- sections[].id — deterministic string, e.g. "experience-0"
- sections[].heading — section heading text
- sections[].headingStyle — TextStyle (see below)
- sections[].items[].id — deterministic string, e.g. "experience-0-item-0"
- sections[].items[].title, titleStyle — optional
- sections[].items[].subtitle, subtitleStyle — optional
- sections[].items[].bullets[].id — deterministic, e.g. "experience-0-item-0-bullet-2"
- sections[].items[].bullets[].text — raw bullet text without leading "•" or "-"
- sections[].items[].bullets[].style — TextStyle

TextStyle fields (required for DOCX reconstruction):
- fontName: string — "Calibri", "Times New Roman", etc.
- fontSize: number — in half-points (OOXML unit): 24 = 12pt
- bold: boolean
- italic: boolean
- color: string — hex "#1a1a1a"
- lineSpacingPt?: number — line height in points (optional)
- spaceBefore?: number — paragraph spacing before in points (optional)
- spaceAfter?: number — paragraph spacing after in points (optional)

RewrittenBullet fields:
- id: string — matches Bullet.id in ResumeStructure
- original: string — unchanged original bullet text
- rewritten: string — AI-rewritten bullet text
- approved: boolean — defaults to false (client-side approval state)

AnalysisResult fields:
- score: number min 0 max 100 — labeled "keyword alignment estimate", NOT "ATS score"
- gaps: string[] — JD keywords absent from resume
- rewrites: RewrittenBullet[]
- resumeStructure: ResumeStructure — echoed for stateless round-trip to /api/generate

CRITICAL: All fields must be JSON-serializable primitives (string, number, boolean, plain objects, arrays).
No Date objects, Buffer, class instances, or undefined values — these are silently dropped by JSON.stringify.

**packages/types/src/resume.ts**:
```typescript
import { z } from 'zod';

export const TextStyleSchema = z.object({
  fontName: z.string(),
  fontSize: z.number(),              // half-points (OOXML): 24 = 12pt
  bold: z.boolean(),
  italic: z.boolean(),
  color: z.string(),                 // hex "#1a1a1a"
  lineSpacingPt: z.number().optional(),
  spaceBefore: z.number().optional(),
  spaceAfter: z.number().optional(),
});

export const BulletSchema = z.object({
  id: z.string(),                    // deterministic: "experience-0-item-0-bullet-2"
  text: z.string(),                  // raw text without leading "•" or "-"
  style: TextStyleSchema,
});

export const SectionItemSchema = z.object({
  id: z.string(),
  title: z.string().optional(),
  titleStyle: TextStyleSchema.optional(),
  subtitle: z.string().optional(),
  subtitleStyle: TextStyleSchema.optional(),
  bullets: z.array(BulletSchema),
});

export const SectionSchema = z.object({
  id: z.string(),
  heading: z.string(),
  headingStyle: TextStyleSchema,
  items: z.array(SectionItemSchema),
});

export const ResumeStructureSchema = z.object({
  meta: z.object({
    pageWidth: z.number(),
    pageHeight: z.number(),
    marginTop: z.number(),
    marginBottom: z.number(),
    marginLeft: z.number(),
    marginRight: z.number(),
  }),
  sections: z.array(SectionSchema),
});

export type TextStyle = z.infer<typeof TextStyleSchema>;
export type Bullet = z.infer<typeof BulletSchema>;
export type SectionItem = z.infer<typeof SectionItemSchema>;
export type Section = z.infer<typeof SectionSchema>;
export type ResumeStructure = z.infer<typeof ResumeStructureSchema>;
```

```typescript
import { z } from 'zod';
import { ResumeStructureSchema } from './resume.js';
import { RewrittenBulletSchema } from './bullet.js';

export const AnalysisResultSchema = z.object({
  score: z.number().min(0).max(100),
  gaps: z.array(z.string()),
  rewrites: z.array(RewrittenBulletSchema),
  resumeStructure: ResumeStructureSchema,
});

export type AnalysisResult = z.infer<typeof AnalysisResultSchema>;
```
