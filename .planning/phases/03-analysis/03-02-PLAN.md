**2. tokenize(text: string): string[]**
- Lowercase
- Replace all non-alphanumeric characters with space: `.replace(/[^a-z0-9\s]/g, ' ')`
- Split on whitespace
- Filter: keep only tokens with length > 2 that are not in STOP_WORDS

**3. extractResumeTokens(resume: ResumeStructure): Set<string>**
- Collect all text from: `resume.header[].text`, `resume.sections[].heading`, `resume.sections[].items[].title` (if defined), `resume.sections[].items[].subtitle` (if defined), `resume.sections[].items[].bullets[].text`
- Return `new Set(texts.flatMap(tokenize))`

**4. export function analyzeResume(resume: ResumeStructure, jobDescription: string): AnalysisResult**
- `const resumeTokens = extractResumeTokens(resume)`
- `const jdTokens = Array.from(new Set(tokenize(jobDescription)))` — deduplicate before computing
- `const matched = jdTokens.filter(t => resumeTokens.has(t))`
- `const score = jdTokens.length === 0 ? 0 : Math.round((matched.length / jdTokens.length) * 100)` — divide-by-zero guard mandatory
- `const gaps = jdTokens.filter(t => !resumeTokens.has(t))` — already deduplicated from the Set step
- Return `{ score, gaps, rewrites: [], resumeStructure: resume }` — rewrites must be explicit empty array, never omitted
